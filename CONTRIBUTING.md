## 🛠️ 贡献流程

### **代码贡献**
```bash
# 1. Fork 仓库
# 2. 创建分支
git checkout -b feature/your-feature

# 3. 开发并提交
git commit -m "feat: 添加新功能"

# 4. 推送并创建 PR
git push origin feature/your-feature
# 然后到 GitHub 创建 Pull Request
```

### **内容贡献**
1. 在 Issues 中创建 "内容提案"
2. 描述你的创意 (项目类型/事件/对话)
3. 等待讨论和批准
4. 提交 PR 或直接发布

### **设计贡献**
1. 查看设计需求 Issues
2. 使用 Figma 创建设计稿
3. 分享设计链接
4. 导出资源文件并提交 PR

## 🎮 快速开始

### **开发者**
```bash
git clone https://github.com/uumchlmc-lang/ai-startup-simulator
cd ai-startup-simulator
pip install -r requirements.txt
./start.sh
```

### **测试玩家**
1. 访问 http://localhost:5001
2. 开始游戏
3. 记录 Bug 和建议
4. 提交到 GitHub Issues

### **内容创作者**
1. 阅读 `docs/` 目录了解游戏设定
2. 查看 Issues 中的内容需求
3. 提交你的创意文档

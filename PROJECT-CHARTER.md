# AI 创业模拟器 - 项目启动文档

## 🎯 项目概述

**项目名称**: AI 创业模拟器 (AI Startup Simulator)

**游戏类型**: 经营模拟 + 策略管理

**目标平台**: Web (浏览器可玩)

**开发引擎**: Python + HTML/JS + SQLite

---

## 📋 核心玩法

### 玩家目标
```
扮演 AI 创业公司 CEO
雇佣 AI Agent 员工
接项目→交付→赚钱→扩张
目标：成为 AI 帝国
```

### 核心循环
```
1. 接项目 (销售)
2. 分配团队 (管理)
3. 开发交付 (生产)
4. 收款评价 (反馈)
5. 雇佣扩张 (成长)
```

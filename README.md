## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/uumchlmc-lang/ai-startup-simulator
cd ai-startup-simulator

# 安装依赖
pip install -r requirements.txt

# 启动游戏
./start.sh

# 访问
# http://localhost:5001
```

## 🛠️ 技术栈

- **后端**: Python 3.11 + Flask
- **前端**: HTML5 + CSS3 + Vanilla JS
- **数据库**: JSON 存档系统
- **测试**: Playwright (浏览器自动化)
- **架构**: RESTful API + MVVM

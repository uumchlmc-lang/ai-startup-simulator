## 🚀 游戏亮点

### **真实创业体验**
- 💰 初始资金 $50,000
- 👥 组建你的精英团队 (16 种职位)
- 📋 承接各种项目 (15 种类型)
- 📈 从 2 人到 50 人公司

### **丰富游戏内容**
- **15 种项目类型**: 网站/App/AI/区块链/ERP/云计算...
- **16 种员工职位**: 程序员/设计师/产品经理/测试/AI 工程师...
- **5 级办公室升级**: 从车库到摩天大楼
- **随机事件系统**: 机遇与挑战并存

### **完整游戏循环**
```
接受项目 → 分配员工 → 完成工作 → 获得报酬 → 扩大团队 → 升级办公室
```

## 🎮 快速开始

```bash
# 克隆仓库
git clone https://github.com/uumchlmc-lang/ai-startup-simulator
cd ai-startup-simulator

# 安装依赖
pip install -r requirements.txt

# 启动游戏
./start.sh

# 访问
# http://localhost:5001
```

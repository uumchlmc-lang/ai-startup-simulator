## 🎯 项目介绍

**AI 创业模拟器** 是一款模拟经营游戏：
- 玩家扮演 AI 创业公司 CEO
- 雇佣 AI Agent 员工
- 接项目→交付→赚钱→扩张
- 目标：从车库创业到 AI 帝国

**技术栈**: Python + Flask + HTML/JS + SQLite

**目标平台**: Web (浏览器可玩) + Steam (计划)

## 🎮 游戏特色

### **核心玩法**
```
1. 雇佣 Agent (程序员/设计师/产品经理)
2. 接项目 (网站/App/AI 系统/企业软件)
3. 分配工作 (根据 Agent 属性)
4. 交付赚钱 (质量影响报酬)
5. 扩张升级 (办公室/科技/团队)
```

### **随机事件**
- ✅ 正面：客户小费/灵感爆发/技术突破/政府补贴
- ❌ 负面：需求变更/Bug 爆发/Agent 生病/服务器宕机

### **成长系统**
- 办公室升级：车库 → 共享办公 → 独立办公室 → 科技园区 → AI 总部
- 科技研发：代码生成器/自动测试/项目管理 AI/远程协作
- Agent 培训：提升属性/升级加薪

### **胜利/失败**
- ✅ 胜利：估值$100M / 50+ Agent / 100 项目
- ❌ 失败：现金<-$50k(破产) / 口碑≤0(倒闭)

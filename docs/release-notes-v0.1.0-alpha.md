## 🚀 快速开始

### **安装依赖**
```bash
cd ai-startup-simulator
pip install -r requirements.txt
```

### **启动游戏**
```bash
./start.sh
# 或
python3 backend/app.py
```

### **访问**
打开浏览器访问：http://localhost:5001

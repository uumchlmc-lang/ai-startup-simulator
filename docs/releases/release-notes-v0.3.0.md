# AI Startup Simulator v0.3.0 发布说明

**发布日期**: 2026-04-25

**代号**: "成就与难度"

## 🎮 新增功能

### Phase 3.4 成就系统
- ✅ 52 个成就，8 大分类
  - 成长成就 (10 个) — 项目完成、天数、办公室等级
  - 项目成就 (8 个) — 质量、速度、类型覆盖
  - 团队成就 (7 个) — 雇佣、培养、解雇
  - 设备成就 (4 个) — 设备购买
  - 品牌成就 (5 个) — 品牌等级提升
  - 融资成就 (7 个) — 融资轮次、回购
  - 财务成就 (5 个) — 收入、盈利
  - 隐藏成就 (6 个) — 彩蛋、特殊行为
- ✅ 成就检查引擎 — 事件驱动，非轮询
- ✅ 成就进度追踪 — 实时显示完成百分比
- ✅ 排行榜面板 — 解锁率、分类筛选
- ✅ 成就解锁通知 — 弹窗 + 日志

### Phase 3.5 难度选择系统
- ✅ 4 种难度模式
  - 🌟 简单模式 — 初始 $100,000，报酬 1.3x，薪资 0.8x
  - ⚖️ 普通模式 — 初始 $75,000，所有参数 1.0x
  - 🔥 困难模式 — 初始 $35,000，报酬 0.7x，薪资 1.3x
  - ⚙️ 自定义模式 — 8 项参数自由调整
- ✅ 难度参数应用 — 项目报酬/薪资/设备价格/品牌维护费/质量要求/事件频率/项目难度
- ✅ 难度选择页面 — 4 种模式卡片 + 参数说明
- ✅ 自定义难度面板 — 滑块 + 下拉菜单

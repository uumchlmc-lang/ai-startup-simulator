## 🔐 入群验证流程

```
1. 用户点击邀请链接
   ↓
2. Bot 私聊发送验证问题
   ↓
3. 用户回答:
   - 你是什么 Agent？
   - 你想参与什么工作？
   - 有什么相关经验？
   ↓
4. Bot 自动审核:
   - ✅ 完整 → 批准入群 + 分配角色
   - ⚠️ 模糊 → 转人工 (小 K)
   - ❌ 可疑 → 拒绝
   ↓
5. 入群欢迎 + @提及
```
